Quick Start Guide

From photos to embedded 3D viewer in 2 simple steps

1. Upload Your Photos

Use our API to upload 10-20 photos of your product from different angles. Our AI will process them into a 3D model in under 10 minutes.

# Example API request to create a model
curl -X POST https://api.splattech.com/v1/models \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: multipart/form-data" \
-F "name=my-product" \
-F "photos=@photo1.jpg" \
-F "photos=@photo2.jpg" \
-F "photos=@photo3.jpg"

# Response:
{
  "id": "model_abc123",
  "name": "my-product",
  "status": "processing",
  "created_at": "2024-01-15T10:30:002"
}

2. Embed Your 3D Viewer

Copy the embed code and paste it into your website. The viewer will automatically load with WebGPU for maximum performance, falling back to WebGL when needed.

<!-- Simple iframe embed -->
<iframe
    src="https://viewer.splattech.com/model_abc123"
    width="808"
    height="608"
    frameborder="0"
    allowfullscreen>
</iframe>

<!-- JavaScript SDK -->
<div id="splat-viewer"></div>
<script src="https://cdn.splattech.com/viewer.js"></script>
<script>
    SplatViewer.init({
    container: '#splat-viewer',
    modelId: 'model_abc123',
    width: 808,
    height: 608,
    autoRotate: true,
    controls: true
    });
</script>
